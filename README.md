# ❤️ Heart Disease Prediction

## 📌 Overview
This project predicts whether a patient is at risk of heart disease using a Machine Learning model.

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 📊 Features
- Predicts heart disease risk
- Interactive web interface
- Real-time predictions
- Responsive UI

## 📁 Dataset
UCI Heart Disease Dataset

## 🤖 Model
K-Nearest Neighbors (KNN)
